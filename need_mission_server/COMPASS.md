MongoDB Compass Connection

1) Open MongoDB Compass.
2) In the connection dialog, paste this URI:
   mongodb://127.0.0.1:27017/need_mission

3) Click "Connect". You should then see the `need_mission` database appear after you submit a form or run the seed.

Collections:
- memberships
- contactmessages

Tip: Use the Filter bar, e.g. { "type": "Institutional" } on `memberships`.
